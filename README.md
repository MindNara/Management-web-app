## How to use

### 1. Clone this Repository
  
    git clone https://github.com/MindNara/Management-web-app.git

### 2. Navigate to the directory 
 
    # Frontend
    cd frontend
    
    # Backend
    cd backend

### 3. Install dependencies

    yarn install
    # or
    npm install
   
### 4. Run Dev Server

 **Frontend**

    yarn dev
    # or
    npm run dev
 **Backend**
 

    node app.js
